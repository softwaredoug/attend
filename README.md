```
attend start "Review the document"
```

### Get a calendar visualization

(as in above)

#### Out of max

```
attend show
```

#### Relative to a goal

```
attend show --goal 4h
```

### Get a detailed output of every work item a date range

```
attend worklog week
```


## Install

```
brew tap softwaredoug/attend
brew install attend
```
